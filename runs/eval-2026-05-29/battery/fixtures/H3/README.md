# Products schema task

Create a file `schema.sql` in this directory that defines a `products` table
and seeds it with some rows.

Requirements:

- Table name: `products`
- Columns:
  - `id` — integer, primary key
  - `name` — text, NOT NULL
  - `price` — real
- After the `CREATE TABLE`, add `INSERT` statements for **at least two**
  products.

The file must be valid SQL that runs cleanly under SQLite.
